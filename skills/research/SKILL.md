# Research

Gather trusted evidence, synthesize it, and convert it into an engineering decision.

## Workflow

1. Define the decision the research must support.
2. Search primary sources first: official docs, specs, papers, release notes, source repos.
3. Use secondary sources only for context or community wisdom.
4. Record dates, versions, and source links for time-sensitive claims.
5. Compare options against the user's constraints.
6. End with a recommendation, caveats, and next verification step.

## Source Priority

Prefer:

- Official documentation
- Standards and specs
- Maintainer-written posts
- Peer-reviewed or primary research
- Source code and issues

Treat SEO blogs, copied tutorials, and unsourced rankings as weak evidence.

## Output Shape

- Question
- Sources checked
- Findings
- Recommendation
- Risks or unknowns
