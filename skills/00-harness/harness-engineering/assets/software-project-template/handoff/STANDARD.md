# Handoff Standard

## Purpose

Compress current task context so another agent or future session can continue safely.

## Report Location

Write handoffs to `handoff/` using:

```text
YYYY-MM-DD-<short-topic>.md
```

Each handoff must include:

- Objective.
- Current state.
- Key files.
- Decisions made.
- Commands run and results.
- Open issues.
- Next step.
